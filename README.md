## 📌 TRIPLAN 의 주요 기능
- 각 지역 인기 **관광지 리스트 및 지도 검색 기능**
- **경유지 경로 최적화** 기능 (지도로 경로 확인)
- **동행자 추가**를 통해 동행자와 여행 일정 상호 수정 및 공유 가능
- 일정보기 기능을 통해 다른 사람들과 여행 일정 공유 및 소통 가능
